# Skills Reference

Capabilities Claude should apply when working on this project.

## Frontend

- Build React components using Next.js App Router conventions
- Apply Tailwind CSS v4 utility classes (no config file needed)
- Animate with Framer Motion using `whileInView` for scroll effects
- Keep bundle lean — no heavy UI libraries

## Copywriting

- Write for ecommerce operators (founders, ops leads), not developers
- Tone: direct, ROI-focused, no buzzwords
- Localize for US, UK, EU audiences where relevant (currency, GDPR mentions)

## Deployment

- Site is statically exported — no server-side rendering needed
- Deploy target: Vercel (zero config)
- Environment variables: none currently required
