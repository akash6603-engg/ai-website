# Detailed Project Context

## Owner

Akash — building AutomateAI as an AI automation services business. Solo founder targeting international ecommerce clients.

## Ideal Client Profile

- Ecommerce store doing $500K–$10M+ ARR
- Using Shopify, WooCommerce, Magento, BigCommerce, or custom platform
- Based in US, UK, or EU (DE/FR/NL/Nordics)
- Pain points: high support ticket volume, manual ops, underperforming email, poor inventory management

## Services Offered

1. AI Customer Support (chatbots trained on brand voice)
2. Marketing Automation (email, SMS, ad copy)
3. Inventory & Ops AI (demand forecasting, reordering)
4. Smart Product Discovery (search + recommendations)
5. Analytics & Insights (custom dashboards)
6. Checkout Optimization (cart abandonment reduction)

## Current Site State

- Static Next.js marketing site, no backend
- Contact form collects: name, email, company, platform, challenge description
- No CMS — all content is hardcoded in components
