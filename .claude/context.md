# Project Context

## Target Markets

| Market | Notes |
|--------|-------|

| Europe | Multi-language, GDPR compliance required, DE/FR/NL/Nordics |

- No database, no auth — purely static marketing site
- Contact form is not yet wired to a backend (placeholder submit handler)
