# Reusable Prompts

Prompts to use when asking Claude for common tasks on this project.

## Add a new page section

```
Add a new section called [NAME] between [SECTION_A] and [SECTION_B] in src/app/page.tsx.
It should follow the same whileInView animation pattern used in the other components.
The section id should be "[id]" for navbar anchor linking.
```

## Write marketing copy

```
Write copy for the [SECTION] section targeting ecommerce founders in [US/UK/EU].
Tone: confident, direct, no fluff. Focus on ROI and time savings.
Avoid buzzwords like "cutting-edge", "revolutionary", "leverage".
```

## Debug a build error

```
The Next.js build is failing with this error: [PASTE ERROR]
The project uses Next.js 16, TypeScript strict mode, Tailwind CSS v4, and Framer Motion.
```
